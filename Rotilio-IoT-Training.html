Rotilio Training Program



Basic Training

During the training we will retain the following topics:

  • Rotilio Maker components: sensors, actuators, communication interface, WiFi and 3G connections
  • WiFi configuration
  • 3G configuration
  • Rotilio Maker and Rotilio Pro functions
  • Web/Mobile Application
  • Tutorial how to turn on and off eletrical loads with Rotilio Maker
  • Tutorial how to turn on and off eletrical loads with Rotilio Pro
  • Tutorial of how to receive info from Rotilio
  • Main differences between Rotilio Maker and Rotilio Pro
  • Creating a mini application

Skills

The following basic skills are required:

  • Familiarity with browser, mobile app and WiFi networks
  • No programming capacity required even if it might facilitate the understanding of some parts
  • Familiarity with electrical fittings for the hardware part
