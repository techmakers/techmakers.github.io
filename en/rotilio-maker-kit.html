ROTILIO THERMO

An open source thermostat which is customizable, cost effective and smart. This tutorial will teach you how to build a thermostat for the home or the office heating system, remotely controlled via a mobile app running on iOS or Android.

You will learn how to:

  • Write functional software which is expandable and ready for the latest programming standard
  • Control physical objects via app and Web browser
  • Integrate multiple Web services to create a functional open ecosystem

Moreover, because in the IoT reality software interacts with the physical world, you will also learn how to:

  • Write software that takes into account system errors
  • Protect system from any error, be it environment reading related or human error
  • Start mechanisms only when necessary and save energy
